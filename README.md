# pwa-install-app

Detect native app and install with the help of PWA. If native app is already installed PWA will open the app. It also add PWA to home screen on `Windows/Linux/MacOS`.

## Features

- [with-typescript](https://github.com/zeit/next.js/tree/canary/examples/with-typescript)
- [Next.js](https://nextjs.org/)
- [Preact.js](https://preactjs.com/)
- [tailwindcss](https://tailwindcss.com/)
- [postcss](https://postcss.org/)
- [with-styled-components](https://github.com/zeit/next.js/tree/canary/examples/with-styled-components)

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
